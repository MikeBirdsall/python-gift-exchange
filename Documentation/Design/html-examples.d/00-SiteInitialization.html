Wish List Site Initialization

Welcome to the Wish List Site initialization page.

Below is a form to input needed information to initialize the site for use. This will set up enough user information for your user which can can finish later
and allow the program to define the Master Administrator and initial user group. The program will set up your user along with default group and initial user group,
calendar and wish list. After this is done you will be transferred to the Site Administration Page to begin populating the site with users, groups, other
administrators and calendar information. After you have set up more users and groups you may need to edit your user information with such information as spouse, children, ...
