Mike and Ed's New Wish List Prototype Pages

Edit Date: Thursday March 221, 2019

     Introduction

     Michael and I are working on a newer updated version of Michael's perl script website for Wish List used for Christmas and other gift giving events for the family. As part of this we are presenting various web pages for people
     to review and see if they like or have ideas of changes or alternates. So we are looking for feedback on how you like this or suggestions for changes, additions and deletions.

     Note1: These initial pages are designed for the Personal Computer and may not display well on tablets or smart phones. We hope to get to that shortly after we have pages for things working for computers.
     Note2: These are not an exhaustive listing of all the possible pages as we are still working on others and revisions of these.
     Note 3: Most of the examples are based on Edward Birdsall. If you belong to more than one "Clan" then you would see information on all clans. Currently clans used for examples are Birdsall and Kirkup.
     Note 4: The buttons on the pages are not active yet so we will be adding tabs to show off the pages

     Processes and Views

  • Home Page
    •       Every user when they sign in start at their home page. The example is Edward Birdsall's Home Page. From here the user will go to other pages and return here as desired to continue working on things. You will notice a number of changes on the pages and how you get around the website.
  • Calendar Page
    • Here the calendar page is for March 2019 and has a number of different calendar events displayed.
  • Update my List
    1. We are trying to have links able to goto alternate wish sites like Amazon and debating on how that would work.
    2. Checking my list and editing those items that need update or deletion
      3. To change an item or delete an item on the list the Edit button and would go to a web page which would show item detail and allow for changing things or delete it. After you edit the suggestion and hit the Submit Suggestion button or hit the Delete Suggestion button you will go to a page to verify that you wish to do this. If when you got to verify and you spot something wrong you can hit the Return to Editing button to correct and verify again.
    3. Adding items not on the list
      4. If you wish to add an item there is an Add Item button which will send you to an Add item page where you can add an item and then verify it to get it on your list.
  • Make up a new shopping list
    1. Step 1: Goto shopping list generation, pick who to be on the list and create it
    2. Step 2:Now for those people on the list select or reserve the item then Create Shopping List
      Note: If you only selcted one person then you would get a single person listed and not multiple as shown above
      • Adding an item to someone else's list
      • Adding notes to someone else's list
  • Got out shopping and buy things
    1. Marking an item bought is done by clicking on BUY on either your shopping list or that person's wish list
    2. Unreserving an item is done by going to your shopping list and clicking on Discard button for that item.
    3. Unbuying an item is done by going to your shopping list and clicking on Undo-Purchase button for that item.
    4. Checking the person's list while shopping is done by going to your shopping list or that person's wish list.
