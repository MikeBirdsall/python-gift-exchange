Mike and Ed's New Wish List Prototype Pages

Edit Date: Thursday April 9, 2019

Introduction

Michael and I are working on a new updated version of the Wish List used for Christmas and other gift giving events for the family. We would like to get feedback and any suggestions on the layout and how things are organized, and how they will work. To start that, we've mocked up some web pages (they don't work yet) that give a first draft idea of how the look, along with some bit of information about how you can move between them to get things done.

This page is an index to all the mock-ups we have, if you want to like for them.

A companion webpage, linked here, is provided to walk you through the use of the various pages, more in the order you would use them.
